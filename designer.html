📥 How to Install in Hammerspoon
1. Install Hammerspoon: Download from hammerspoon.org if you haven't already
2. Configure workspace: Drag apps from the library onto monitors and configure splits as needed
3. Copy generated script: Use the "Copy Script" button - it includes both Gentle (⌘⌥⌃G) and Full Reset (⌘⌥⌃W) modes
4. Open Hammerspoon config: Click Hammerspoon menu bar icon → "Open Config" or edit ~/.hammerspoon/init.lua
5. Paste into config: Add the generated code to your init.lua file
6. Reload Hammerspoon: Click Hammerspoon menu bar icon → "Reload Config"
7. Use your workspace: Press ⌘⌥⌃G for gentle arrangement or ⌘⌥⌃W for full reset!
